# globalo

Top-level object detector.

## Usage

```js
var root = globalo();
```

- Node.js: `global`
- Browsers: `window`
- Titanium + Alloy: `Alloy.Globals`
- Titanium  `Ti.App`

## Installation

npm:

```bash
npm install globalo
```

component:

```bash
component install vesln/globalo
```

## Tests
